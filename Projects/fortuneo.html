Fortuneo Machine Learning Project (KNN)

Binary classification on a real-world, highly imbalanced dataset (~3% positives).
Objective: detect clients likely to subscribe to an insurance policy.

1. Dataset & Challenges

  • Highly imbalanced dataset (~3% positives)
  • Mixed numerical & categorical variables
  • Noise affecting distance-based models
  • Main metric: Recall on class 1

2. Preprocessing Pipeline

  • Train/Test split before preprocessing
  • Label Encoding + StandardScaler
  • Outlier removal
  • Rebalancing using SMOTEENN
  • Class weights tested to improve recall

3. KNN Model

  • KNN on rebalanced data
  • GridSearchCV for optimal k & distance metric
  • Bias–variance analysis

4. Evaluation

  • Confusion Matrix
  • Precision / Recall / F1-score
  • Prediction scatter

5. Insights

  • SMOTEENN improves minority detection
  • GridSearchCV strengthens model stability
  • Model tuned to prioritize recall
